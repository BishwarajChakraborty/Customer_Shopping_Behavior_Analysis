**Project Title: Customer Shopping Behavior Analysis**

**Project Overview:**
This project provides an end-to-end data analysis of customer purchasing patterns to uncover insights into revenue drivers, product performance, and customer loyalty. The analysis was performed using a multi-tool approach: SQL for data extraction and transformation, Python for processing, and Power BI for interactive visualization.

**Key Technical Highlights:**
1. Database Management: Built and managed a structured database using SQL to handle 3,900+ customer transaction records.
2. Advanced Querying: Leveraged CTEs (Common Table Expressions) and Window Functions (like DENSE_RANK) to perform complex data segmentation and ranking.
3. Interactive Dashboards: Developed a Power BI report to visualize key business metrics and trends across different demographics.

**Core Insights & Analysis:**
1. Revenue Segmentation: Analyzed total revenue and average spend by Gender and Subscription Status to identify high-value customer segments.
2. Customer Loyalty Mapping: Categorized the customer base into New, Returning, and Loyal segments based on purchase history to help tailor marketing strategies.
3. Product Performance: Identified the top-performing products by category and those with the highest customer review ratings.
4. Promotional Impact: Measured the effectiveness of discounts by calculating the "Discount Rate" for various product items.
5. Logistics & Behavior: Compared spending habits across different Shipping Types (Standard vs. Express) and analyzed the correlation between purchase frequency and subscription likelihood.

**Tools & Technologies Used:**
1. SQL: Data cleaning, complex joining, and business logic implementation.
2. Python (Jupyter Notebook): Exploratory Data Analysis (EDA) and data manipulation.
3. Power BI: Data modeling and creation of interactive visual insights.
4. Dataset: Comprehensive customer attributes including demographics, location, payment methods, and seasonal trends.

**Key Takeaway:**
This project demonstrates how data-driven insights can identify which products drive the most engagement and which customer segments are most likely to contribute to long-term revenue growth.
